## Установка
___
Скачиваем [Archiver-1.0-SNAPSHOT.jar](https://github.com/MorozovArYu/Archiver/raw/main/resources/Archiver-1.0-SNAPSHOT.jar).

В командной строке прописываем команду:

    java -jar Archiver-1.0-SNAPSHOT.jar
